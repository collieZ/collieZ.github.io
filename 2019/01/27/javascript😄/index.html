微信公众号接入介绍

当我们注册了微信公众开发平台后，我们选择左下方的开发中的开发者工具，然后选择其中的公众号测试账号。使用测试账号我们可以不用花时间去申请公众号，且测试账号所有api接口都能供我们调用，不用受到正式公众号的各种限制。

  1. 进入测试账号后，我们可以看到

其中appid 、appsecret、token是接入微信服务器所需要的，token是由自己定义。

  1. 按照官方文档定义我们要接入微信的服务器需要如下几步：

    1. 获取微信服务器Get请求的参数 signature、timestamp、nonce、echostr
    2. 将token、timestamp、nonce三个参数进行字典序排序
    3. 将三个参数字符串拼接成一个字符串进行sha1加密
    4. 开发者获得加密后的字符串可与signature对比，标识该请求来源于微信

    看着这一堆操作不免觉得麻烦，而且网上也已经有了一大堆代码能实现接入请求，我们也不用重复造轮子，学习一下思想即可。（这里说明一下微信服务器接入验证这一块是get请求，往后的都是post请求。）

  1. 为了方便开发，同时减轻工作量，这里我们使用了两个nodejs的微信库，wechat、wechat-api 。使用这两个库之后我们就不需要再写一些重复性的东西了，甚至不需要关注如何接入验证，如何请求，应答微信服务器，我们只需要关注我们的需求就行。
  1. 所以我们接下来只需要提供微信公众号的appid、appsecret、token（自己制定的），将其提供给api中的函数就可以与微信服务器建立连接啦～（具体代码上的实现我们后面再了解）

接入前的一些准备

  • 接入微信公众号的服务器之前我们需要配置好我们的服务器，并在我们服务器上讲node.js环境搭建好，以及一些依赖的安装。因为开始我们可能并没有拥有备案域名的服务器，所以这里我们可以利用自己的pc作为服务器，使用内网穿透工具将本地的服务穿透到外网。所以，接下来我们会做一下几点：
  1. 在本地pc（服务器）上搭建node.js环境，安装express框架。
  2. 安装一些依赖，以及wechat api。
  3. 进行内网穿透

  • 首先我们先构建node.js的运行环境 ，首先来了解一下node.js

    Node.js®是在Chrome的V8 JavaScript引擎上构建的JavaScript运行环境。Node.js使用事件驱动的非阻塞I / O模型，使其轻量且高效。

可以理解为node.js就相当于一个在服务器端运行的js环境，这使得用js进行后端开发成为了可能，关于node.js的更多介绍可以自行去了解。

  1. 安装node.js。 可以直接在node.js官网下载对应系统版本的安装包安装，也可以在命令行中输入：(windows可以在gitbash中输入，dos不行).

wget https://nodejs.org/dist/v8.11.2/node-v8.11.2.tar.gz

Pass:具体的版本自己选择，一般以最新最稳定版本为准。

然后进入到下载目录解压:

tar -xzvf node-v0.12.5.tar.gz

进入到解压后的目录：

cd node-v0.12.5

执行配置脚本进行编译预处理：

./configure

编译源码

make && make install

pass： 由于不同系统安装方式不同，不同的安装包也有不同的安装方式，具体情况具体分析，自行网上查找解决办法。（上面的方法需要配置好gcc、c++的编译环境，所以嫌麻烦的话可以用其他方法。）

至此node环境就搭建完毕了。我们可以在命令终端中输入node. 看是否能进入node的环境，如出现这样

就说明安装成功了。

  1. 接着我们需要安装node.js的express框架。执行如下指令安装（npm是node.js的一个包管理系统 类似python的pip）

npm install -g express

进行全局安装，windows环境好像要设置环境变量，具体可以参考

  1. 安装好之后输入:

express xxx

xxx是你要创建的项目文件名。 也可以直接在某个文件夹中直接输入express 构建工程文件夹。 完了我们可以发现在当前文件夹下生成了这么几个文件\文件夹

根据提示我们在express生成的文件夹下输入

npm install

安装一些依赖包

  1. 接着就该安装两个微信api wechat-api 和 wechat，依然是在express生成的文件夹下输入：

    npm install wechat -save // wechat 安装

    npm install wechat-api // wechat-api安装

好了，我们已经安装好了微信的api了。
